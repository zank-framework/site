We appreciate submissions and tips or feedback (info@zank.ir). We create zank for create static website from database driven website. Here's some website example to use zank framework:

  • News Portals
  • Ads Portals
  • Shopping websites
  • CMS based websites
  • Commenting systems

The best way to contribute is to fork the repo, then make a new branch and submit a pull request. 

$ cd work-directory

$ git clone https://github.com/your-awesome-username/zank.git

$ git branch my_version
$ git push origin my_version
$ git checkout my_version

Then you can start working locally, commit and push as you normally would. When you are happy with your contribution, you can submit a pull request

We will then personally review the request and add it (and your name) to the project
